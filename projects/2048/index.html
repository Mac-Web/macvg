2048
Gabriele Cirulli

About

2048 is a very popular and classic puzzle game. The goal is to merge the tiles with the same numbers until they merge all the way to 2048. Use arrow keys to move all the tiles. If no tiles can be moved anymore, you fail.

Controls

  • Arrow / WASD Keys: Move Number Tiles

Game Information

  • Developer: Gabriele Cirulli
  • Players: 100M+
  • Catagory: Strategy and Puzzle
  • Genre: Number Games
  • Build: HTML5, JavaScript
